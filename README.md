# goshare

### install

```shell
go install https://github.com/CodeCrafters-Community/goshare@latest
```

### user

> add you <$GOPATH/bin> to env

macos

```shell
echo "export PATH=$GOPATH/bin:$PATH:." >> ~/.bash_profile && source ~/.bash_profile
```

### run

```shell
goshare
```

```
goshare --help
```
